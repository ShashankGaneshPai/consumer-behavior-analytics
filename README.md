# 📊 Customer Behavior Analytics

## End-to-End Data Analytics Project using Python, SQL & Power BI

This project demonstrates a complete **end-to-end Data Analytics workflow** on a retail customer behavior dataset. It follows the industry-standard analytics process—from understanding a business problem to cleaning data, performing exploratory data analysis (EDA), executing SQL-based business analysis, building an interactive Power BI dashboard, and presenting insights through a professional report and presentation.

The project showcases the practical skills required for a **Data Analyst** role and serves as a portfolio project.

---

## 📌 Overview

The goal of this project is to analyze customer shopping behavior and identify actionable business insights that can help improve decision-making.

The workflow includes:

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business Analysis using SQL
- Interactive Dashboard Creation in Power BI
- Project Documentation
- Presentation Creation

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/ShashankGaneshPai/consumer-behavior-analytics.git
```

### 2. Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn mysql-connector-python
```

### 3. Run the Python Notebooks

Open the notebooks in **Jupyter Notebook** or **Visual Studio Code** and execute them sequentially.

### 4. Import Data into MySQL

- Create a database in MySQL Server.
- Import the cleaned dataset.
- Execute the SQL scripts located in the `SQL` folder.

### 5. Open the Power BI Dashboard

Open the `.pbix` file using **Power BI Desktop** and refresh the data source if required.
